# Weather App

## 📌 Description
This project is a weather web application that fetches real-time weather data using an API.

## 🚀 Features
- Search weather by city name
- Displays temperature, weather condition, and humidity
- Simple and clean UI
- Uses API to fetch live data

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## ⚠ Setup
1. Create an account on OpenWeather
2. Get your API key
3. Replace YOUR_API_KEY in script.js
